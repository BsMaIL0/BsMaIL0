| **Cocktail Sort** | Реализация шейкерной сортировки (модификация пузырька с двунаправленным проходом). Алгоритм проходит массив слева направо и справа налево, сокращая количество итераций за счёт исключения уже отсортированных границ | O(n²) в худшем случае, O(n) для почти отсортированных массивов | [🔗 Посмотреть](https://github.com/BsMaIL0/myCocktailSort) |
